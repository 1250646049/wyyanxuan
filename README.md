 这个项目是去年写的，只写了前端，用到的技术栈是Vue全家桶、mintUI、axios，没有实现后端接口，全部是抓包至网易商场App接口。
 
实现了挺多的功能的，比如首页、商品列表、商品分类、下单、购物车、个人中心、值得买功能模块等，小的功能模块，UI还原的还是挺漂亮的（不要脸的我 哈哈哈哈），因为没有设计图，所以只能凭照感觉去布局，学生朋友哪来应付作业什么的应该是可以的。
